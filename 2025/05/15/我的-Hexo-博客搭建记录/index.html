title: 我的 Hexo 博客搭建记录
date: 2025-05-15 22:50:06
tags:今天在chatgpt的协助之下终于搭建了我的个人博客，怀着新奇和喜悦的心情，现在来开始分享我的Hexo 博客搭建记录，这也帮助我回顾和反思我在这个过程中遇到的困难以及克服经历，帮助我再遇到相似的场景该怎么去解决问题，以及背后的核心逻辑是什么
#为什么想要搭建个人博客？
记录我每天的学习和成长，把每一天清晰化，并且在这个 过程中不断反思和精进自我
#搭建博客的过程，以及遇到的问题
1.下载git bash,第一次下载失败
原因：在安装的最后一步时出现了两个选项：
A.lauch git bash;B.view release notes
由于还是对全英文的环境存在畏难情绪，所以没有认真思考就直接盲目勾选第二个选项，期盼快快安装完成，结果失败了，后来冷静下来安装第二次的时候才认真读了选项，并且再次询问gpt确认，最终安装成功，发现这两个词组我凭借我的词汇量和联想理解起来也没有那么的难
反映出的问题:对于英语的掌握程度远远不够，专有名词稍微一多就开始慌张并且抗拒去阅读，其实是浪费了时间，如果在第一次就仔细检查并且遇到不懂的词汇善于借助翻译器，那么也不会有安装错误的问题
2.安装Node.js,安装再次失败
原因：没有认真读gpt给我的操作指南，忽略了“勾选 add to path ”这个步骤，导致再次失败
反映出的问题：太浮躁，没能逐字逐句的读完就着急忙慌的操作，不够谨慎，不够认真，对这件事情没能抱有100%的尊重，其原因是不够尊重自己，还是不够尊重自己的时间，反思
3.安装 Hexo （顺利）
4.创建个人博客项目文件夹（顺利）
5.修改博客配置文件，失败两次
原因：第一次，找不到根目录，其实在创建的时候就设置创建在桌面，但是我心浮气躁又忽略了这个细节，所以找了好几次；
第二次，我又是“大概”看了一眼gpt的配置文件怎么写的，然后格式错了一堆，git bash都都读不懂；又改了一次，还是错的，最后是直接复制粘贴gpt原话才终于对了。
反映出的问题：
写到这里我对自己非常无语，其实又是相同的问题，就这样一次又一次的出现在我生命中，我真的还有很长一段路要走，实在是太浮躁了。
6.更换主题，非常之困难，一个小时都没有弄好，心态差点炸了
原因：第一次，是由于网络访问的问题，导致文件下载受阻，git.bash识别失败；
第二次，更换下载方法，由gpt直接给我压缩包，然后下载解压；
结果就这样按照步骤一步一步走，却每次都是运行错误，开始还以为是重命名的问题，但是尝试了很多遍还是失败，我此时已经非常烦躁，但是我心里已经下了决心，不就是换个主题，真的有那么困难吗？我必须把它解决了，今天这个主题换不好我就不睡觉了。怀揣着这样的决心，我决定冷静下来，一步一步的分析对照到底是哪个环节出来问题，最终，在我换了多个解决方案后终于发现，gpt给我的压缩包文件本来就是残缺的，所以就注定一直用这个压缩包就是解决不了问题!我恍然大悟，于是和gpt沟通让他重新给我发压缩包，此时恰逢我的gpt当前的模型免费对话权限用完了，此时的模型已经不能直接给我发它处理好的压缩包，所以对话框一直出错，我察觉这个问题后引导他重新帮助我，最后在github的官网下载到了正确的压缩包，最终更换主题成功了！
心路历程：急切–失落–烦躁–下定决心–冷静–认真仔细–喜悦（成功了）
#反思
通过这次梳理我看到了自己目前的大问题：太浮躁！不够尊重自己。我已经非常深刻的意识到了自己的问题，同时我也看到自己身上的潜能，我也并不是不能静下心来把事情做好，只要有一点点耐心，那么我会不断复制粘贴这一点点的耐心，在每一次的事情中锻炼，不断复利，不允许放弃，我坚信自己能一点点进步，就让博客见证我的进步之旅吧！不求一步登天，只求点滴，只求每次一点点的进步。
